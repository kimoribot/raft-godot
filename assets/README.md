# Raft Godot - 3D Assets

## Project Structure

```
assets/
├── models/
│   ├── characters/     # Player, sharks, NPCs
│   ├── environment/    # Islands, rocks, vegetation
│   ├── items/          # Weapons, tools
│   └── props/          # Barrels, crates, debris
├── textures/          # PNG textures
└── materials/         # Godot materials (.tres)
```
